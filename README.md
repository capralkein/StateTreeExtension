# StateTreeExtension
Small plugin to extend the StateTree plugin. Unreal engine 5.1.1

Evaluators
   1.  Attribute update

Tasks
   1. Move to
   2. Rotate to
   3. Set focus
   4. Environment query
   5. Switch senses in state
   6. Wait availability of abilities
   7. Activate ability
   8. Apply gameplay effect

CancellableAsyncAction
   1. Wait ability end
   2. Wait AI message
